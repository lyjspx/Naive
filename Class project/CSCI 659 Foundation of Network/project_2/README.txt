Please, run the program by the following order

Run physical_wire.exe first, with command: physical_wire.exe wire_port
Run data_link_layer.exe, with command: data_link_layer.exe wire_addr wire_port data_port_1
Run data_link_layer.exe, with command: data_link_layer.exe wire_addr wire_port data_port_2
Run network_layer.exe, with command:network_layer.exe data_addr_1 data_port nickname_1
Run network_layer.exe, with command:network_layer.exe data_addr_2 data_port nickname_2

The program is ready to communicate.

After finish communication, enter 'EXIT' in two network layers.
